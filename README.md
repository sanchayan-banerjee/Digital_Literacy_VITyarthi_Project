## PROJECT OVERVIEW:
This project focuses on developing digital literacy and professional skills required for students in the modern digital world. It includes creating a digital presentation, building an online portfolio, exploring learning platforms, practicing email etiquette and understanding cyber safety. The project aims to enhance awareness of responsible technology usage and improve communication, productivity and online presence.

## Module Summaries:
- **Task 1: Digital Literacy Awareness Infographic**
Created an infographic-style presentation covering digital literacy, tools for students, safe internet practices and email etiquette.

- **Task 2: Student Digital Portfolio Development**

- **Task 3: Exploration of Coding and Collaboration Platforms**
